# 🎬 MovieWeb

Hey there! 👋  
MovieWeb is a modern movie search application I built to improve my **frontend development skills** using React. It allows users to search for movies, explore trending content, and view detailed movie information. The app is powered by the **TMDB API** and uses **Appwrite as a backend service** to store and track popular searches.

This project helped me practice working with APIs, React components, state management, responsive design, and integrating a backend service into a frontend application.

---

## 💡 What It Can Do

- 🔥 Displays trending movies based on popular searches
- 🔎 Allows users to search movies by title
- 🎬 Shows detailed movie information in a modal
- ⭐ Displays movie posters, ratings, and descriptions
- ⚡ Provides a fast and responsive user experience
- 📊 Tracks popular searches using Appwrite database
- ⏳ Uses debouncing to improve search performance

## 🚀 How To Run It

### 1. Clone the repository

```bash
git clone https://github.com/YASMIN-KEDIR/movieWeb.git
```

### 2. Move into the project folder

```bash
cd movieWeb
```

### 3. Install dependencies

```bash
npm install
```

### 4. Add environment variables

Create a `.env` file in the project root:

```env
VITE_TMDB_API_KEY=your_tmdb_api_key

VITE_APPWRITE_PROJECT_ID=your_appwrite_project_id

VITE_APPWRITE_DATABASE_ID=your_appwrite_database_id

VITE_APPWRITE_TABLE_ID=your_appwrite_table_id
```

### 5. Start the development server

```bash
npm run dev
```
